# Publication

| Field          | Type            | Description             |
|----------------|-----------------|-------------------------|
| authors        | array of string | Authors of publication  |            
| year           | string          |         YYYY            |
| pmid           | ExternalDB      |  PubMed citation        | 
| pmcid          | ExternalDB      |  EuropePMC citation     |
| doi            | ExternalDB      |  DOI citation           |
| ucsc_id        | ExteralDB       |  UCSC citation          |
