# OrganismGroupMetadata

### GroupingMethodMetadata
GroupingSourceMetadata is an instance of [ValueSetMetadata](./metadata.md) and describes how the grouping was created.


### Examples

```json
{
  "grouping_method": {
    "accession_id": "grouping_method.consortium",
    "value": "consortium",
    "label": "Consortium",
    "definition": "Consortium defined",
    "description": "This grouping was defined by a consortium"
  }
}
```
